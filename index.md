# Tempura Studio

iOS / Web 向けのアプリを個人開発で提供しています。

## アプリ

### mymap

ドラクエ風 fog of war マップアプリ。歩いた場所だけ地図が解放されます。

- [プライバシーポリシー](/mymap/privacy/)

### Cert Rogue 知識のダンジョン

資格学習をローグライク RPG で。SRS で忘却タイミングに合わせて出題します。

- [プライバシーポリシー](/certrogue/privacy/)
- [サポート](/certrogue/support/)
