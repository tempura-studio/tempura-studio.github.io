# Cert Rogue 知識のダンジョン プライバシーポリシー

**最終更新日**: 2026-06-04

本プライバシーポリシーは、Tempura Studio (個人開発) が提供する iOS アプリ **「Cert Rogue 知識のダンジョン」** (以下「本アプリ」) における、ユーザー情報の取り扱い方針を定めるものです。本アプリをご利用いただく前に、必ず本ポリシーをお読みください。

## 要約

- 本アプリは、ユーザーを識別するためのアカウント登録を一切要求しません。
- 学習履歴・進捗・設定はすべて端末ローカルに保存され、開発者のサーバには送信されません。
- iCloud 同期を有効にした場合、データはユーザー本人の iCloud (Apple 社が運営する Private Database) にのみ保存されます。
- 広告は表示しません。トラッキング目的の SDK は組み込んでいません。
- 課金処理は Apple および RevenueCat が標準仕組みで行い、本アプリは決済情報を取得・保管しません。

## 1. 取得する情報の種類

### 1-1. 端末ローカルに保存される情報

本アプリは、ゲームを継続的にプレイできるようにするため、以下の情報を **ご利用端末内のローカルストレージ (localStorage / IndexedDB)** に保存します。これらは、開発者を含む第三者には一切送信されません。

- ダンジョン進捗 (到達フロア・クリア状況)
- SRS (間隔反復学習) アルゴリズム用の出題履歴・正誤履歴
- 取得済み装備・コイン残高・図鑑の解禁状況
- 音量・操作モードなどのアプリ設定
- 無料/有料コンテンツのアンロック状態 (キャッシュ用)

### 1-2. iCloud (CloudKit Private Database) に保存される情報

iOS 端末で iCloud にサインインし、かつ本アプリの iCloud 同期機能が有効な場合、上記の学習データは Apple 社が運営する **iCloud Private Database (Private DB)** にも保存されます。

- Private DB は、サインインしている Apple ID のユーザー本人のみがアクセスできる領域です。
- 開発者を含む第三者は、Private DB の内容を閲覧・取得することはできません。
- 同期されるデータの内容は「1-1.」と同じ範囲です (個人を識別する情報は含まれません)。
- iCloud 同期を無効にしたい場合は、iOS の「設定」 > Apple ID > iCloud > Cert Rogue をオフにしてください。

### 1-3. 課金に関する情報

本アプリの追加コンテンツ購入には、Apple の StoreKit と、課金管理プラットフォーム **RevenueCat** を利用します。

- 決済情報 (クレジットカード番号等) は本アプリでは取得・保管しません。Apple 側で処理されます。
- RevenueCat には、購入が成立した事実 (どの商品が購入されたか) と、無作為に生成された匿名 App User ID のみが送信されます。氏名・メールアドレス・端末識別子等は送信しません。
- RevenueCat のプライバシーポリシーは [https://www.revenuecat.com/privacy/](https://www.revenuecat.com/privacy/) を参照してください。

### 1-4. 取得しない情報

本アプリは、以下のいずれの情報も取得・収集しません。

- 氏名、メールアドレス、電話番号、住所などの個人を直接識別する情報
- 位置情報 (GPS)
- 連絡先、写真、カレンダー、マイク、カメラへのアクセス
- 広告識別子 (IDFA) およびトラッキング目的のデバイス情報
- ブラウザ Cookie / Web ビーコン (アプリ内で広告・トラッキング SDK を使用していないため)

## 2. 情報の利用目的

ローカルおよび iCloud に保存される学習データは、以下の目的でのみ利用されます。

- ゲームの進捗を継続して保存・復元するため
- SRS アルゴリズムに基づき、忘却タイミングに合わせて問題を出題するため
- 同じ Apple ID の複数の端末で進捗を同期するため (iCloud 同期有効時のみ)

## 3. 第三者提供

本アプリは、ユーザーから取得した情報を、本人の同意なく第三者に提供することはありません。課金処理に関する Apple および RevenueCat への情報送信は、本サービスの提供に必要な範囲に限定されます。

## 4. 広告について

本アプリは広告を表示しません。広告ネットワークの SDK は組み込まれていません。

## 5. データの削除方法

- 端末ローカルのデータを削除したい場合は、iOS の「設定」アプリから本アプリをアンインストールするか、アプリ内の「設定」 > 「データ初期化」をお使いください。
- iCloud に保存されたデータを削除したい場合は、iOS の「設定」 > Apple ID > iCloud > 「ストレージを管理」 > Cert Rogue から削除してください。

## 6. 子どもの利用について

本アプリは学習アプリとして設計されており、特定の年齢層に限定したサービスではありません。13 歳未満の方が利用される場合は、保護者の同意のもとでご利用ください。13 歳未満の方から、保護者の同意なく個人を識別する情報を収集することはありません (そもそも本アプリは個人を識別する情報を取得しません)。

## 7. セキュリティ

端末ローカルのデータは、iOS が標準で提供するサンドボックス機構により保護されます。iCloud に保存されるデータは、Apple のインフラ上で暗号化されて保管されます。

## 8. 本ポリシーの変更

本ポリシーの内容は、法令の変更やアプリ機能の追加に伴い、予告なく改定される場合があります。重要な変更を行う場合は、アプリ内またはサポートページにて告知します。最新版は本ページで常時公開します。
